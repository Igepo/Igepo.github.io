The Wall VR

The Wall VR is a VR game developed under Unity. The player has to survive waves of zombies using his bow and different types of arrows or explosive elements.

I was responsible for the creation of different tools such as explosive barrels to help the player defeat the wave of zombies.

MazeAR

Individual project made with Unity using the Vuforia development kit. It is a project usable on Android. Also, I used Blender to model the maze.

The maze is projected on a book that represents the image that the application recognizes. The objective is to get a marble out of the maze in a given time.

Star Rush

Star Rush is a multiplayer school project. This game has been developed in javascript using the "socket.IO" library. This project taught me to :

- Create a multiplayer browser game.

- Create a Login/Register system linked to a database.

- Deploy a game with a database in order to have it tested by my classmates.
